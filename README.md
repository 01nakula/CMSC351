# CMSC351
University of Maryland CMSC351 (Algorithms) Spring 2021 Lecture Notes.

## Course Topics
* Quadratic Sorting Algorithms
* Summations
* Mergesort
* Recurrences (Integer Multiplication)
* Heapsort
* Quicksort
* Sorting in Linear Time
* Medians and Order Statistics
* Graphs and Trees
* Minimum Spanning Trees
* Dijkstra's Algorithm
* Introduction to NP-completness
* Introduction to Parallel Algorithms

Taught by Professor [Clyde Kruskal](http://www.cs.umd.edu/~kruskal/) with notes compiled by Neelam Akula.
